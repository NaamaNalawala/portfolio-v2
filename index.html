SmartLocker Payment & Hardware System (Bag Bunker)

Systems & Hardware

Led end-to-end development of the Bag Bunker SmartLocker system, transforming a fragile prototype into a production-ready, payment-enabled hardware-integrated platform. The initial system faced payment inconsistencies, locker state failures, hardware edge-case issues, and lacked structured error handling, logging, and recovery mechanisms.

Solution: Architected and implemented the complete application logic using C# and WPF, integrating Nayax payment systems with secure transaction validation and real-time locker state synchronization. Designed robust locker allocation algorithms, failure recovery flows, and fallback handling for hardware disruptions. Introduced structured logging, transaction traceability, retry policies, and defensive programming to eliminate deadlocks between payment confirmation and locker activation. Optimized performance for real-time scoreboard-style live updates across multiple lockers while maintaining data integrity. Coordinated hardware, UI, and backend layers to function as a cohesive system.

Impact: Delivered a stable, scalable SmartLocker solution operating reliably in live environments. Eliminated payment-locker mismatches, reduced manual operational intervention, improved transaction visibility, and strengthened system resilience. Established clean architecture, documentation, and maintainable patterns to support future expansion, admin panel development, and hardware upgrades.

Automation

Solution: Designed governance framework with naming standards, centralized logging, dependency mapping, and clear ownership. Rebuilt flows with retry logic, failure notifications, and rollback procedures.
